# 五子棋游戏 (Gobang)

一个简单的五子棋游戏实现，支持命令行游玩。

## 功能特点

- ✨ 支持 15x15 标准棋盘（可自定义大小）
- 👥 双人对战模式
- 🎯 自动判断胜负（五子连珠）
- 🖥️ 命令行界面，简单易用
- 📝 中文界面和提示

## 快速开始

### 命令行版本

```bash
# 运行游戏
python3 gobang.py

# 或赋予执行权限直接运行
chmod +x gobang.py
./gobang.py
```

### 游戏规则

1. 黑棋 (X) 先手，白棋 (O) 后手
2. 双方轮流落子
3. 先将五子连成一线（横、竖、斜）的一方获胜
4. 棋盘填满且无胜负则为平局

### 操作说明

- **落子**: 输入坐标，如 `7 7` 表示在第7行第7列落子
- **退出**: 输入 `quit`、`exit` 或 `q` 退出游戏
- **棋盘大小**: 启动时可输入自定义大小（推荐15）

## 技术实现

- 纯 Python 实现，无外部依赖
- 使用二维数组表示棋盘
- 胜负判断算法：检查四个方向（横、竖、两对角线）
- 时间复杂度：每次落子 O(1) 检查
